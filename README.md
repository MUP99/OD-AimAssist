# COD AimAssist

🔫 A desktop application that provides aim assist features for Call of Duty games.

## 🎯 Features
- Multiple aim assist levels (Lite, Normal, Middle, Super)
- Color-based enemy detection
- ADS (Aim Down Sight) tracking
- Clean GUI with PyQt5

## 🖥️ How to Use
1. Download the EXE file from the GitHub Actions artifacts
2. Run `COD_AimAssist.exe` on Windows
3. Choose your aim assist mode and enjoy!

## 💼 Built With
- Python 3.11
- PyQt5
- OpenCV
- PyAutoGUI
